# EngelHeartWare - Official Website 🌐

This repository contains the source code for the official website of **EngelHeartWare**, the indie development studio behind **Swipe In** and **Skyline Explorer**. 

Check out the live site here: [https://www.engelheartware.de](https://engelheartware.de)]

## 🏢 About the Site
This site serves as the main portfolio and landing page for all company projects. It includes:
* **The Main Homepage:** Information about the developer and studio.
* **Project Subpages:** Dedicated landing pages, privacy policies, and support resources for individual apps.

## 📱 Featured Projects
If you are looking for the actual source code of the applications featured on this website, please visit their dedicated repositories:

* 👉 **[[Swipe In] iOS Repository](https://github.com/engelheartware/SwipeIn)**
* 👉 **[[Skyline Explorer] iOS Repository](https://github.com/engelheartware/Skyline)**

## 🛠 Tech Stack
* HTML5 / CSS3 / Vanilla JavaScript

## 🚀 Deployment
This static website is deployed via **Netlify**. 
Any changes pushed to the `main` branch are automatically built and published to the live site.
