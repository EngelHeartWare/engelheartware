Datenschutz

Deine Daten bleiben auf deinem Gerät.

Swipe In sammelt, überträgt oder teilt keine persönlichen Daten. Alles bleibt auf deinem Gerät — immer.

  • Keine Analyse- oder Tracking-SDKs
  • Keine Werbenetzwerke
  • Kein Konto erforderlich — niemals
  • Gespeichert mit Apples Core Data Framework
  • Funktioniert vollständig offline
  • Daten verlassen dein Gerät nur, wenn du sie selbst exportierst

Datenschutzerklärung

Zuletzt aktualisiert: April 2026

Welche Daten sammelt Swipe In?

Swipe In sammelt, überträgt oder teilt keine persönlichen Daten. Alle Informationen, die du eingibst — Zeiteinträge, Aktivitäten, Orte und Einstellungen — werden ausschließlich auf deinem Gerät mithilfe von Apples Core Data Framework und UserDefaults gespeichert.

Teilt die App Daten mit Dritten?

Nein. Swipe In enthält keine Werbe-SDKs, Analyse-Frameworks oder Tracking-Bibliotheken von Drittanbietern. Keine Daten verlassen jemals dein Gerät, außer du exportierst sie ausdrücklich selbst über die CSV-Exportfunktion.

Welche Daten werden lokal gespeichert?

  • Deine Zeiteinträge (Startzeit, Endzeit, Aktivitätslabel, Ortslabel)
  • Deine benutzerdefinierten Aktivitäts- und Ortslisten
  • App-Einstellungen (Akzentfarbe, Darstellungsmodus, ausgewähltes Icon)
  • Tutorial-Status (abgeschlossen oder nicht)

CSV-Export

Wenn du die Exportfunktion nutzt, wird eine CSV-Datei auf deinem Gerät generiert und über das iOS-Teilen-Menü bereitgestellt. Es liegt in deiner Verantwortung zu entscheiden, wohin du diese Datei sendest oder speicherst. Die App ist danach nicht mehr involviert.

Datenschutz von Kindern

Swipe In sammelt keine Daten von Nutzern, einschließlich Kindern unter 13 Jahren.

Änderungen an dieser Richtlinie

Sollte sich diese Richtlinie in einem zukünftigen Update ändern, wird die aktualisierte Version hier mit einem neuen Datum veröffentlicht.

Häufige Fragen

Ist Swipe In wirklich kostenlos?

Ja — keine Werbung, kein Premium-Abo. Wenn die App dir Zeit spart, freue ich mich über einen Kaffee via Ko-Fi.

Funktioniert sie offline?

Vollständig. Alles wird lokal auf deinem Gerät gespeichert. Zu keinem Zeitpunkt ist eine Internetverbindung erforderlich.

Wie exportiere ich meine Daten?

Gehe zu Verlauf → tippe auf das Export-Symbol. Eine CSV-Datei wird auf deinem Gerät erstellt und über das iOS-Teilen-Menü bereitgestellt.

Kann ich einen vergangenen Eintrag bearbeiten?

Ja — tippe auf einen beliebigen Eintrag im Verlauf, um Startzeit, Endzeit, Aktivität oder Ort zu bearbeiten.

Wird es eine Android-Version geben?

Aktuell nicht geplant — iOS steht im Fokus. Das kann sich in Zukunft ändern.

Wo werden meine Daten gesichert?

Deine Daten sind automatisch im Standard-iCloud-Backup deines iPhones enthalten. Eine dedizierte iCloud-Synchronisierung folgt demnächst.
